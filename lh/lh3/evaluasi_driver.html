Analisis dan Penjelasan

Halaman Login

Dari hasil evaluasi didapat beberapa user yang menanyakan mengenai form daftar. Kami tidak memiliki form daftar karena, pendaftaran driver tidak lewat aplikasi tetapi langsung ke kantor pusat. Alasan kami tidak adanya pendaftaran melalui aplikasi karena, kami harus mengetahui profil calon driver lebih lengkap dan rinci melalui surat – surat Negara yang berlaku. Seluruh user yang mencoba setuju dalam hal tersebut.

Halaman Daftar Order (Home)

Dari hasil evaluasi didapat beberapa user yang tidak begitu pahap tentang button ON/OFF. Button ini berfungsi sebagai penanada dan untuk mengatur bahwa Driver tersebut siap menerima Order atau bealum. Saran dari user adalah untuk memberi keterangan dibawah button tersebut. Adapun beberapa user yang kebingungan dengan icon keranjang yang kami buat, yang berfungsi sebagai button menuju Halaman Riwayat Order, user menyarankan untuk mengganti icon dengan icon “Jam”.

Halaman Riwayat Order

Dari hasil evaluasi didapat seluruh partisipan user tidak tau bagaimana caranya kembali ke Tampilan Daftar Order(Home). Seluruh partisipan tidak tahu bahwa icon FoodJas adalah button menuju Tampilan Daftar Order(Home). Seluruh partisipan menyarankan untuk menambahkan icon Home/Rumah pada navigasi bagian atas aplikasi, tetapi tidak menghilangkan fungsi dari icon FoodJas yang mengantarkan user kembali ke Tampilan Halaman Order (Home).

Implikasi Hasil Terhadap Rancangan

Dari hasil evaluasi secara umum dapat disimpulkan bahwa rancangan yang dibuat sudah dapat menyelesaikan masalah dan membantu user dalam mengeksekusi task yang dibutuhkan. Adapun masukan-masukan yang diungkapkan user bersifat minor, sehingga dapat diimplementasi tanpa mengganggu hasil analisis yang ada.

Kritik Terhadap Rencana Evaluasi

Halaman Login
  • Menambahkan fungsional Daftar, yang berfungsi untuk mengantarkan user halaman website yang berisi cara mendaftar menjadi Driver FoodJas, yang tidak dilakukan pada aplikasi.
Halaman Daftar Order (Home)
  • Menambahkan keterangan pada bagian bawah button ON/OFF dengan kalimat “Siap Menerima Order” pada saat ON dan “Belum Siap Menerima Order” pada saat OFF
  • Mengganti icon keranjang pada yang mengantarkan user menuju halaman Riwayat Order menjdai icon jam.
Halaman Riwayat Order
  • Menambahkan icon Home/Rumah pada bagian navigasi aplikasi
